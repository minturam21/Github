# Git Basics – Complete Notes

## 1. What is Git and Why It's Used

**Git** is a distributed version control system.

### Why Git exists:
- Tracks changes in code over time
- Lets multiple people work on the same project safely
- Allows reverting to previous versions if something breaks
- Keeps a history of who changed what and why

### Key Idea:
Git is not just backup — it's **change tracking + collaboration control**
